# Implementation Plan - Lazo Toiture

## Goal
Build a high-performance web platform for **Lazo Toiture & Énergie**.

## Tech Stack
Next.js 14, Tailwind, Shadcn/UI.

## Steps
1. Initialize Project (Next.js 14 + TS + Tailwind).
2. Setup UI (Shadcn, Fonts).
3. Build Components (Header, Wizard, Footer).
4. Build Pages (Home, Services, Zones).
5. Implement SEO (Schema, Metadata).
